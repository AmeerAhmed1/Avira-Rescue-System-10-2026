## 🎮 How to Use
1. **Create the Rescue Media**: Run the executable and follow the on-screen prompts to create a bootable USB drive.
2. **Boot from USB**: Restart your computer and boot from the USB drive (may require changing boot order in BIOS).
3. **Start the Scan**: Once booted, Avira Rescue System will load; select "Start Scan" to initiate a full system scan.
4. **Review Results**: After scanning, review detected threats and choose actions (clean, quarantine, or delete).
5. **Reboot and Verify**: Remove the USB drive, reboot your computer, and verify that the system is clean.
